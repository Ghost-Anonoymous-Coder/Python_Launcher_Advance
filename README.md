# 👻 Ghost-Mod (v3.0.0.5)
> **The Ultimate Python Launcher Application for Mass Storage Devices**

---

## 📝 Description
**Ghost-Mod** is a powerful, lightweight Python launcher designed to aggregate, manage, and execute Python scripts directly from external mass storage devices. Whether you are managing local files or external drives, Ghost-Mod centralizes your workflow into a single, secure environment.

---

## 📦 System Requirements

📂 **Supported Environments:**
*   **Android:** Fully operational via **Termux**
*   **Desktop:** Compatible with **Windows, macOS, and Linux**

🔧 **Dependencies:**
*   Please refer to the `requirements.txt` / `pip.txt` file for the mandatory Python modules.
